📍 Address → Lat/Lon Geocoder

Upload a CSV or XLSX file with an address column (header in row 1, case-insensitive). Each address is geocoded and exported with new lat, lon, match_level and matched_address columns, in the same format you uploaded. Pick the country below (default Israel; Hebrew supported, e.g. דיזנגוף 50, תל אביב). By default only exact matches are returned; enable the fallback option to let messy addresses degrade to street/city. Check match_level to judge each result.

Geocoding uses the free Nominatim (OpenStreetMap) service, which is rate-limited to ~1 request/second. A hard-to-match address may try several variants (exact → cleaned → street/city → city), so it can take a few seconds; identical queries are cached. Large files take a while — budget ~1 second per attempt.
